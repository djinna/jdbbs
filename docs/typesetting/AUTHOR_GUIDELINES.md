# Author Guidelines for Manuscript Preparation

## Overview

Your manuscript will be professionally typeset for the format(s) you've chosen:
- **EPUB only** — Digital reading on phones, tablets, e-readers
- **EPUB + Print** — Both digital and physical books
- **Print only** — Physical books only (paperback/hardcover)

This guide will help you prepare your manuscript for smooth conversion to your chosen format(s).

## Quick Start: The Golden Rule

**Use Word styles consistently.** Don't manually format text—apply styles instead.

❌ **Don't:** Select text → Bold button
✅ **Do:** Apply a character style like "Strong" or paragraph style like "Heading 1"

## Getting Your Custom Template

After submitting your manuscript transmittal, we'll generate a custom Word template specifically for your book. This template will include:

1. **Standard styles** everyone needs (see Required Styles below)
2. **Custom styles** specific to your manuscript (if you have special formatting needs)
3. **Your book's typography** settings (fonts, sizes, spacing)

To request special styles, note them in your transmittal under "Special Characters" or "Instructions."

## Required Styles (Available in Every Template)

### Paragraph Styles

| Style Name | When to Use | Notes for EPUB | Notes for Print |
|------------|-------------|----------------|-----------------|
| **Normal** | Regular body paragraphs | Reflowable | Auto-indented |
| **First Paragraph** | After any heading or break | No indent | No indent |
| **Heading 1** | Chapter titles | Creates navigation | Forces new page |
| **Heading 2** | Major sections | Creates navigation | Adds space above |
| **Heading 3** | Subsections | May create navigation | Adds space above |
| **Block Quote** | Extended quotations | Can keep formatting | Indented, may be italic |
| **Code Block** | Terminal output, code | Monospace, scrollable | Monospace, smaller |
| **Verse** | Poetry, song lyrics | Preserves line breaks | Smaller, italic, indented |
| **Epigraph** | Opening quotations | Preserved | Right-aligned or indented |
| **Section Break** | Scene/section divisions | Ornamental break | Ornamental break |

### Character Styles

| Style Name | When to Use | Notes |
|------------|-------------|-------|
| **Emphasis** | Stressed words | Becomes italic |
| **Strong** | Important text | Becomes bold |
| **Code** | Inline code/commands | Monospace font |
| **Book Title** | Titles of books | Italic in both formats |
| **Article Title** | Titles of articles | Quotes in print, may be styled in EPUB |
| **Foreign** | Non-English words | Italic, may have language tag |
| **Small Caps** | Acronyms, special text | True small caps in print |

## Format-Specific Considerations

### If You're Publishing EPUB Only

**What's preserved:**
- Colors (though most e-readers show grayscale)
- Hyperlinks (internal and external)
- Most font variations
- Tables (will reflow on small screens)

**What to avoid:**
- Fixed positioning (text boxes, sidebars)
- Multi-column layouts
- Page number references ("see page 47")

### If You're Publishing Print Only

**What works well:**
- Consistent use of styles
- High-contrast formatting (bold, italic)
- Standard layouts (chapters, sections)

**What's normalized:**
- All colors → black
- All fonts → house fonts (unless specially requested)
- Hyperlinks → plain text
- Complex layouts → simplified

### If You're Publishing Both EPUB + Print

Follow the stricter print guidelines. We'll enhance the EPUB version where appropriate while keeping the print version clean.

## Section Breaks

To indicate a scene break or section division:

**Preferred method:**
- Insert → Horizontal Line (Word will show a line, we'll convert to ornamental break)

**Also accepted:**
- Type three asterisks: `* * *` (must be on its own line)
- Apply "Section Break" paragraph style

**Your template will use:** _[ornament style specified in your book spec]_

## Lists

### Numbered Lists
Use Word's numbering feature (not manual "1. 2. 3.")
- **EPUB:** Proper HTML lists, accessible
- **Print:** Consistent formatting

### Bulleted Lists
Use Word's bullet feature (not manual "• " or "- ")
- **EPUB:** Proper HTML lists
- **Print:** Consistent bullets

## Special Characters

| Character | How to Type | EPUB | Print |
|-----------|------------|------|-------|
| Em dash — | Alt+0151 (Win) or Option+Shift+- (Mac) | ✓ | ✓ |
| En dash – | Alt+0150 (Win) or Option+- (Mac) | ✓ | ✓ |

| "Curly quotes" | Word auto-corrects these | ✓ | ✓ |
| © ® ™ | Insert → Symbol | ✓ | ✓ |

## Images

### For EPUB
- **Format:** JPEG or PNG
- **Resolution:** 1200px wide minimum
- **Placement:** Will be centered
- **Caption:** Use caption style

### For Print
- **Format:** TIFF or high-res JPEG
- **Resolution:** 300 DPI minimum
- **Placement:** Discuss with your editor
- **Color:** CMYK for color books

### For Both
- Provide highest quality available
- Include alt text for accessibility
- Number figures if referenced in text

## What to Avoid

### In All Formats
- ❌ Multiple spaces for alignment
- ❌ Tabs for indentation (except code blocks)
- ❌ Empty paragraphs for spacing
- ❌ Manual page breaks (except part divisions)
- ❌ All caps for emphasis (use Small Caps style)

### Additional for EPUB
- ❌ Text boxes or floating elements
- ❌ Columns (except tables)
- ❌ Fixed measurements ("2 inches wide")
- ❌ Page number references

### Additional for Print
- ❌ Colored text for emphasis
- ❌ Hyperlinks as primary content
- ❌ Fonts not in our library (ask first)

## Custom Styles

If your manuscript needs special formatting:

1. **Describe it** in your transmittal
2. **Show examples** in your manuscript with comments
3. **We'll add custom styles** to your template

Common custom styles we support:
- Letter/email excerpts
- Text messages/chat logs  
- Newspaper headlines
- Handwritten notes
- Computer interfaces
- Marginalia or side notes

## Checklist Before Submission

- [ ] Used the provided Word template
- [ ] Applied paragraph styles (not manual formatting)
- [ ] Used "First Paragraph" after all headings/breaks
- [ ] Marked section breaks properly
- [ ] Images are high-resolution with captions
- [ ] Special formatting needs noted in transmittal
- [ ] Spell check and grammar check complete
- [ ] Track Changes turned off, all changes accepted
- [ ] Comments resolved or contain formatting notes only

## Getting Help

- **Template issues:** Contact your production editor
- **Style questions:** Add a comment in your Word doc
- **Special needs:** Note in your transmittal

Remember: Consistent use of styles makes both formats better. When in doubt, use the style that best describes your content's *purpose*, not its appearance.
