WHAT IT IS
----------
A web app built from Jenna's old .xls production schedule
(Prepr_ArtofGig.xls). Go backend, SQLite DB, vanilla JS SPA.
Password-protected, publicly accessible, runs as systemd service.

FEATURES WORKING
----------------
• 3 views: Timeline (Gantt), Table, Budget
• 31 tasks imported from the original .xls spreadsheet
• Task editing (click any row → modal with all fields)
• Password protection per project
• "Make New" — duplicates a project with:
  - New name, client slug, project slug, start date
  - All dates shifted proportionally
  - Budgets zeroed, statuses reset
• Client/project URL routing: /{client}/{project}/
  - Art of Gig lives at /vgr/aog/
  - Pattern ready for additional clients/books
• Light/dark mode toggle (☀️/🌙 button, persists in localStorage)
• Readable timeline with month labels + week tick marks
• Runs as systemd service (survives reboots)
• Admin project list at root /
• Public access configured
